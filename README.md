# Distributed printing System

Formal Methods of Software Engineering project done in Overture tools using VDM++ formal language. Project consists of simulating a company with a distributed printing system with a sample of features: 

 * Distributed printing queue depending on the type of document (black&white, color, A3, A4) 
 
 * Payment with balance 
 
 * Printers placed in different places/clients

 * Solve malfunctions reported / send employees to fix those 
 
 * Reports with several metrics, etc.
